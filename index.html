Instructions for creating your own JivoChat design label

If you do not like the default widget which was created by JivoChat, you can make your own. Let's assume that the address of your site is http://mysite.com. Instructions:

  1. Copy the folder "jivosite", which is in the archive to the root directory of your site - http://mysite.com/jivosite

  2. Verify that the script source code is available at "http://mysite.com/jivosite/jivosite.js"

  3. Add the following code to every page of your site between the <head> and </head> tags:

  4. Make sure that the site has already registered the base JivoChat code (you can find it in your account on the first page). The order of the scripts on your website is not important

  5. Replace the image tag in the directory "http://mysite.com/jivosite/images/" with the ones that you would like to see on the website. The Photoshop source file can be found in the archive.

  6. Change the CSS file height div to match the height of the images (the rows marked with comments)

The widget is now ready. The logic of the label is described in the code comments in the file jivosite.js. If you have any questions - please contact us at info@jivochat.com , or go to the chat (chat on this page, by the way true - you can write to us: )
